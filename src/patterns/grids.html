Responsive

Responsive behaviors are provided by the Responsifier. The following illustrates how it can be used to to create a responsive grid. Resize your browser to stack/unstack grid items. To stack grid items, add the flex-dir-col class.

      data-range="0,1200" 
      data-add="flex-dir-col"></div>
