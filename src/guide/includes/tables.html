Tables

Tables are for displaying tabular data and therefore it's important to leverage the default syntax.

Columns

Standard column based tables should look like this

Heading 1 Heading 2 Heading 3
Element 1.1 Element 1.2 Element 1.3
Element 2.1 Element 2.2 Element 2.3
Element 3.1 Element 3.2 Element 3.3

Pivot

A table can be "pivoted " by adding the molecule pivot creating a row based layout of the same table.

Heading 1 Heading 2 Heading 3
Element 1.1 Element 1.2 Element 1.3
Element 2.1 Element 2.2 Element 2.3
Element 3.1 Element 3.2 Element 3.3

Wrap

An optional wrap class is available to wrap the table data for smaller screens.

Heading 1 Heading 2 Heading 3
Element 1.1 Element 1.2 Element 1.3
