About the ER Frontend Design System

FEDS (Frontend Design System)

FEDS is a Design System developed for Emergency Reporting for the purpose of standardizing it's use of CSS and Javascript. It's goal is to create well a defined toolkit of reusable standards based styles and components that leverage native platform functionality.

Toolkit Includes:

  • Living Style Guide
  • Implementation Patterns
  • Javascript Components
  • Prescriptive Workflows
  • Best Practices & Learning Resources

Key Features:

  • Atomic Design
  • Responsive Design
  • Framework agnostic
  • Intuitive conventions (BEM, OOCSS)
  • Deployment friendly (autoprefixed, optimized, CI friendly)
  • W3C Compliant
