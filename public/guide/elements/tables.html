Tables

Tables are for displaying tabular data and therefore it's important to leverage the default syntax.

Name Phone DOB
Abby 123-123-1234 10/12/76
Bruce 08/26/85 345-678-9101
Catherine 234-567-8910 03/21/95

Pivot

A table can be pivoted by adding the pivot class. This creates a row based layout of the same table.

Name Phone DOB
Abby 123-123-1234 10/12/76
Bruce 08/26/85 345-678-9101
Catherine 234-567-8910 03/21/95

Wrap

An optional wrap class is available to wrap the table data for smaller screens.

In order for wrapping to work, the data-title attribute must be applied to all td's

Name Phone DOB
Abby 123-123-1234 10/12/76
Bruce 08/26/85 345-678-9101
Catherine 234-567-8910 03/21/95

Responsive

Both the pivot and wrap modifiers include media query prefixes.

Auto-wrap

Add the table--wrap class to default to a table that wraps. Add the media query prefixed class tablet:table--unwrap to unwrap the table on devices gte to the tablet width.
