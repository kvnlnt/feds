Pivot

A table can be pivoted by adding the pivot class. This creates a row based layout of the same table.

.table--pivot.mobile:table--pivot.tablet:table--pivot.desktop:table--pivot

Wrap

An optional wrap class is available to wrap the table data for smaller screens. In order for wrapping to work, the data-header attribute must be applied to all td's

Responsive

Both the pivot and wrap modifiers include media query prefixes.

Auto-wrap

Add the table--wrap class to default to a table that wraps. Add the media query prefixed class tablet:table--unwrap to unwrap the table on devices gte to the tablet width.
