Responsifier

The Responsifier class is an DIY implementation of Container Queries. It's purpose is to provide a simple mechanism to add, remove and toggle classes in response to events. Combined with Atoms and Molecules the Responsifier class provides powerful abstractions which:

  • Replaces the need for media queries
  • Eliminates the need for ad-hoc responsive styles and scripts
  • Is framework agnostic
  • Has a small footprint (~5kb)

Usage

                
                    var r = new feds.Responsifier()
                    .add({
                        observer: "#mobile-menu",
                        observable: window,
                        event: feds.Responsifier.RESIZE,
                        range: [0, 799],
                        add: ["show"]
                    })
                    .add({
                        observer: "#mobile-menu",
                        observable: window,
                        event: feds.Responsifier.RESIZE,
                        range: [800, "*"],
                        remove: ["show"]
                    })
                    .init();

Methods

add(options)

Adds a media query to a responsifier instance.

Options
Name Type Example Desc
observer String|[Node List] ".some-button" element to update
observable String|[Node List] "body" element to watch
event Responsifier event constant Responsifier.SCROLL event to watch for (see events for options)
range [number,number|"*"] [0,100] Start and ending ranges. Ending range can optionally be set to "*".
add [string] ["hide"] Array of classes to add
remove [string] ["hide"] Array of classes to remove
toggle [string] ["hide"] Array of classes to toggle
conditions [function] [function(){return true;}] Callback functions to call before making class changes. All must return true.
lock [string] ["hide"] Prevent class from being added, removed, toggled, etc.
unlock [string] ["hide"] Renable class for being added, removed, toggled, etc.

Events

  • Responsifier.SCROLL
  • Responsifier.RESIZE
  • Responsifier.CLICK
  • Responsifier.HOVER
  • Responsifier.TOGGLE
